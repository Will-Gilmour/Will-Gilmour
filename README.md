## What I'm building

### Paper Graph

Scientific literature is a network, but almost every tool we have for navigating it gives you a ranked list instead. I think that's a mistake worth taking seriously.

I built a system that crawls citation data from open APIs, maps papers into a navigable graph using GPU-accelerated force-directed layout, detects thematic communities, and labels them using a locally-hosted LLM. I built a little map while trying this, starting from five seed papers about a niche surgical procedure, it expanded to 700,000 papers spanning neuroscience, biomedical engineering, radiology, and several surgical specialties.

I tested it against a published systematic review and recovered 62% of the included papers in about 30 minutes - and the whole thing runs locally. Indeed once it's been setup the map could be hosted anywhere and you'd be able to get the full benefit just using a browser.
